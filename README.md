Temperature Converter

A stylish and interactive Temperature Converter Web App built using HTML, CSS, and JavaScript. This project allows users to easily convert temperatures between Celsius, Fahrenheit, and Kelvin with a modern UI.

✨ Features
🔄 Multiple Conversions
Celsius → Fahrenheit
Fahrenheit → Celsius
Celsius → Kelvin
🎨 Modern UI Design
Glassmorphism effect
Gradient background
Smooth animations
⚡ Instant Conversion
Displays result immediately on button click
🔁 Swap Function
Quickly switch between Celsius and Fahrenheit
⚠️ Input Validation
Alerts user if input is empty
🛠️ Technologies Used
HTML5
CSS3
JavaScript
